# Edu Navigator

Edu Navigator is a student-focused web application designed to help users explore educational paths, career options, and academic resources in a structured and user-friendly way.

---

## Features

- Provides structured educational guidance  
- Simple and user-friendly navigation  
- Organized content to support student decision-making  
- Helps explore different career and study paths  

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## How to Run

1. Clone or download this repository  
2. Open `index.html` in any web browser  

---

## Project Structure

- `index.html` – Main entry point  
- `src/` – Application logic  
- `routes/` – Routing logic  
- `models/` – Data models  
- `database/` – Database-related files  
- `config/` – Configuration files
